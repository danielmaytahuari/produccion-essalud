# Sistema de Producción Diaria - EsSalud

Sistema de registro de producción diaria para **Tecnólogos Médicos** del Hospital Nacional Alberto Sabogal Sologuren (HNASS).

🔗 **URL:** https://danielmaytahuari.github.io/produccion-essalud

## Características
- 🔐 Login seguro con DNI
- 👤 Registro público de Tecnólogos
- 📊 Registro de producción por sala y turno
- 📈 Reportes mensuales exportables
- 🔑 Panel Admin para gestionar Tecnólogos (CRUD)
- ☁️ Datos en Firebase Firestore

## Tecnologías
- React 18 + Firebase + Tailwind CSS
- GitHub Pages (Hosting)
